Here I use Django and Instaloader to build a website where users can pick winners or find their page ghost followers. First, started a Django project and movied everything to a docker container. Added PostgreSQL and made the needed apps. I also used Django Crispy Forms and Django-allauth, popular 3rd party package.

"Free picker" let's user try the service without singing up and paying, but only returns one winner. A complete URL should be given for the code to find the Short-Code and wait fot it to sing in our Instagram account.

"Winners picker" let's user find many winners from likes, comments and mentions of a post. Here also a complete URL should be given for the code to find the Short-Code. You can have as many winners as you need but It shouln't be higer than participants.

"Ghost follower" returns a list of all the followers, who have never liked a post from that page. It needs a considerable amount of time for the code to calculator therefor, it's not possible to do for accounts with more than 500 followers.
